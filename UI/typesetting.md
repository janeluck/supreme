# 排版准则

> [ant-design前端使用手册](https://os.alipayobjects.com/rmsportal/HTXUgPGkyyxEivE.png)

+ 容器类边距上下为`24px`, 左右为`16px`
+ 空间摆放间距
    - 同类组件之间的间距: `8px`
    - 相关联但不同类的组件间距: `16px`
    - 不关联也不同类的组件间距: `24px`
+ 行高间距/段间距
    - 字体/行高: `18/12`, `21/14`
+ 字号选用范围
    - 主标题: `16px bold #666`
    - 次标题: `14px bold #666`
    - 小标题: `12px bold #666`
    - 正文: `12px  #666`
    - 辅助文字: `12px  #999`
    - 失效文字: `12px  #ccc`
    - 链接文字: `12px  #2DB7F5`
+ iconfont
    - 与文字统一大小
